Your public profile is a single page at `meufolio.dev/{you}`, built entirely from your GitHub data — your bio, your real tech stack, and your projects ranked by impact.

## What's on it

- **Overview** — a short bio pulled from your GitHub profile.
- **Stack** — the languages and tools you actually use, detected from your repos and README badges.
- **Projects** — your repos ranked by impact, so the strongest work shows up first.
- **Experience, Certifications, Languages** — optional sections you fill in yourself, see [Customization](/docs/customization).

## Staying accurate

Because the profile is generated from GitHub, it never goes stale the way a hand-written portfolio does — push new repos or update a README, and the next sync picks it up.
