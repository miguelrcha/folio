### O Folio é gratuito?

Sim — grátis enquanto estiver em beta.

### É open source?

Sim, o código-fonte completo está no [GitHub](https://github.com/miguelrcha/folio).

### Meu token do GitHub está seguro?

Seu token de acesso do GitHub é criptografado em repouso e só é usado no servidor para sincronizar seus repositórios — ele nunca é exposto no seu perfil público.
