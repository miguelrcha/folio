Todo perfil pode ser exportado como um currículo — os mesmos dados, formatados para impressão.

## Exportando

Abra seu perfil e clique em **Ver Currículo**. Isso renderiza um layout dedicado, de uma única página A4, e abre a caixa de diálogo de impressão do seu navegador, pronta para salvar como PDF.

## Por que é separado do perfil

O currículo usa sua própria tipografia e layout, ajustados para caber em uma página — não é apenas uma captura de tela do site. Não há nenhuma etapa de renderização no servidor envolvida, então funciona da mesma forma onde quer que você hospede o Folio.
